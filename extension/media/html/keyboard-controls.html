Keyboard Controls

Creation Controls

alt Create a new entity
alt Create a new value entity
alt Create a new fact relation
alt Use Lasso () tool
alt Use Space () tool
crtl Toggle keyboard help menu
Tab Toggle selection between shapes
alt Step back in selection history
alt Step forward in selection history

When any element is selected, the element can be deleted by pressing the del key or the icon.

Enities / Value Entities

When an entity or value entity is selected the following keyboard shortcuts are available:

Link entity to fact relation
Make entity a substype of other
Change the type of the entity
Change the reference mode of the entity
Create a connected unary fact relation

Fact Relations

When an fact relation is selected the following keyboard shortcuts are available:

Expand the arity of the fact relation
Reduce the arity of the fact relation
Create/remove derived label
Set direction of verbalisation to the left
Set direction of verbalisation to the right
Remove direction of verbalisation
Create uniqueness constraint
Make/remove objectification of relation
